# curlnapi-win32-x64-msvc

Native Node.js addon for curl-impersonate on Windows x64 (MSVC).

## Files
- `curlnapi-node.win32-x64-msvc.node` — native addon
- `libcurl.dll` — runtime dependency
- `zlib.dll` — runtime dependency

## Usage
```js
const { Impit } = require('@wengo/curlnapi-win32-x64-msvc')
```
